# Estudos

## O que estudar, em ordem

### OWASP Top 10 (já está no seu doc) — comece pelo A07:2021 Identification and Authentication Failures e A01:2021 Broken Access Control, que cobrem diretamente esse cenário.
### OWASP Cheat Sheet Series — especificamente "Cross-Site Request Forgery Prevention" e "JSON Web Token for Java" (mesmo sendo Java, a teoria de validação de JWT é universal) — https://cheatsheetseries.owasp.org/
### RFC 6265 (HTTP State Management / cookies) — entender SameSite, Secure, HttpOnly na fonte primária, não só em blog posts.
### Auth0 blog sobre refresh token rotation — é a melhor explicação prática que existe sobre detecção de reuso de refresh token.
### "The Web Application Hacker's Handbook" — se quiser entender como um atacante pensa de verdade, esse livro é a referência clássica. Vai te dar a perspectiva ofensiva que ajuda a antecipar o que você ainda não cobriu defensivamente.
